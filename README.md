# EyeRest - Chrome Extension

A simple, beautiful Chrome extension to help you remember to take regular eye breaks using the 20-20-20 rule.

## Features

*   **Customizable Timer:** Set a timer for 1, 5, or 20 minutes to remind you to take a break.
*   **Full-Screen Break Notification:** When the timer goes off, a full-screen notification appears, prompting you to look at something 20 feet away for 20 seconds.
*   **Soothing Design:** A calming blue gradient theme to make your breaks more pleasant.
*   **Mute Option:** Mute the Text-to-Speech announcement during breaks.
*   **Pause/Resume:** Pause and resume the timer at any time.

## How to Use

1.  **Set the Timer:** Click on the EyeRest icon in your Chrome toolbar to open the popup. Choose a preset time of 1, 5, or 20 minutes.
2.  **Take a Break:** When the timer ends, a new tab will open with a 20-second countdown. Look at an object at least 20 feet away until the countdown finishes.
3.  **Restart:** Click the "Re-start" button or simply close the tab to automatically start the next timer cycle.
4.  **Pause or Mute:** Use the "Pause" and "Mute" buttons in the popup to control the timer and sound.
